# UsageWidget Privacy Policy

Effective date: July 19, 2026

UsageWidget is a self-hosted iOS app. The developer does not operate the
production UsageWidget server and does not collect, sell, rent, or use your
personal data for advertising, analytics, or tracking.

## Data handled by the app

- The server URL and access token you enter are stored in the iOS Keychain and
  shared with the UsageWidget widget through the app's private App Group.
- Usage data is requested from the server you choose and displayed in the app
  and widget. It is not sent to the developer.
- If you enable notifications, the app sends its APNs and widget push tokens to
  your chosen server so that server can request alerts and widget refreshes.
- Camera access is used only to scan a UsageWidget setup QR code. Images are not
  retained or sent to the developer.

UsageWidget contains no third-party advertising or analytics SDKs and does not
track users across apps or websites.

## App Review environment

Apple reviewers may use a temporary isolated review server containing only
synthetic sample data. Push delivery in that environment is simulated. It has
no access to personal production data or configuration.

## Contact

For privacy questions, open a private security report or contact the maintainer
through the [UsageWidget GitHub repository](https://github.com/EdmundLimBoEn/UsageWidget).
